How to sync to widget:
1. Tap "Save journal.json to iCloud"
2. In the share sheet, tap Save to Files
3. Navigate to iCloud Drive → Scriptable
4. Tap Save — widget updates on next refresh

The file contains up to 20 photos (starred first, then newest). Photos are compressed to ~50KB each to keep the file small.
