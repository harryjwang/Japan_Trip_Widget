This version uses your phone's GPS and a built-in sample places dataset, so it can still demo the experience without needing a live places API.
To make it truly dynamic for any city, the next step would be connecting a live places source like Google Places, Foursquare, or OpenStreetMap Overpass.
How it works
1. It asks for your location.
2. It estimates distance to known places.
3. It filters by food, shopping, or attractions.
4. You can save places or open walking directions.
