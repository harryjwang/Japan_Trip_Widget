This version uses your phone's GPS and the OpenStreetMap Overpass API to fetch nearby places live. It needs internet access to search nearby stops.
The results depend on OpenStreetMap tagging, so some areas will have richer results than others. Tap Refresh My Location whenever you move to a new area.
How it works
1. It asks for your location.
2. It sends a live nearby search to Overpass.
3. It groups results into food, shopping, and attractions.
4. You can save places or open them in maps.
